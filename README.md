# Superstore Sales Dashboard (Power BI)

## 📊 Overview
This project is an interactive Power BI dashboard built using the Superstore dataset.

## Features
- Total Sales
- Total Quantity
- Total Profit
- Average Discount
- Sales by Region
- Profit by Category
- Top 10 Products by Sales
- Sales by Segment

## Tools Used
- Microsoft Power BI
- Superstore Dataset
